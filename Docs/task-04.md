### Пояснения по выполнению задания  
  
В приложении подключен Entity Framework, создан контекст базы данных и структура базы данных.  
Используемая база данных - PostgreSQL.  
  
**[Скрипт создания базы данных со всеми миграциями](DbCreationScript.md)**  
(Создан с помощью **dotnet ef**)  
  
---  
    
Созданы CRUD контроллеры для работы с User и Employee сущностями, плюс вся цепочка (Контроллер -> Менеджер -> Репозиторий -> Контекст базы данных)  
Дополнительно то же самое сделано для сущностей Contract и Client.  
  
Методы DELETE для всех моделей сделаны путем выставления флага IsDeleted в объектах, фактического удаления не происходит.  
  
---  
  
В целях оптимизации, для менеджеров запросов к репозиториям сделан общий дженерик интерфейс **IBaseManager<TModel, TRequest>** 
а интерфейсы отдельных менеджеров наследуются от него (по аналогии с интерфейсами репозиториев)
